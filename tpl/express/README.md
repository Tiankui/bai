# Family-Node-Server
### 工程开始步骤

- `git clone`下项目
- 进入项目目录运行`npm install`安装所需依赖
- 再运行`npm link bai`(要求全局安装bai,`npm install -g bai`)
- `node app`运行项目,`bai
run`运行前端监控编译.(建议终端打开两个窗口,此任务不适合`bg`方式使用)
- 项目依赖nodejs
